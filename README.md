# data_visualisation
## Tech Stacks
1. `@nivo/line`
2. `@nivo/pie`
3. `create-react-app` with `@vite`
4. Bootstrap 5
5. ExpressJs
6. Mongose and MongodB
## Commands
### Backend
1. `cd backend`
2. `npm install`
3. `nodemon index`
4.  Load-data by `http://localhost:3000/api/data_save`
5.  Start Frontend
### Frontend
1. `cd frontend/app`
2. `npm install`
3. `npm run dev`
